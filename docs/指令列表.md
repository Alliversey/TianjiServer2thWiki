# 指令列表

|  指令名称   |               指令效果               |                            备注                            |
| :---------: | :----------------------------------: | :--------------------------------------------------------: |
|  `/spawn`   |     将玩家传送到主世界出生点     |                                            |
|  `/money`   |     查看自己的余额     |                                            |
|  `/qs help`   |     箱子商店QuickShop帮助     |                                            |
|  `/co i`   |     开启或关闭方块检查状态    |                                            |
|  `/cmi`   |     CMI基础插件帮助    |                                            |
|  `/cback`   |     返回上次所传送的地点    |                   冷却时间60s                         |
|  `/dback`   |     返回上次所死亡的地点    |                      冷却时间60s                      |
|  `/server`   |     连接其他服务器    |                                            |
|  `/res`   |     领地插件Residence帮助    |                                            |
|  `/sit`   |     坐在方块上    |                                            |
|  `/lay`   |     躺在方块上    |                                            |
|  `/bellyflop`   |     腹部翻转在方块上    |                                            |
|  `/spin`   |     在方块上转动    |                                            |
|  `/crawl`   |     在方块上爬行    |                                            |
|  `/playerstatus`   |     查看延迟与服务器TPS状态    |                                            |
